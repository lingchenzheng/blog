# windows 上常用的快捷键

## 切换窗口

【Alt】 + Tab

## 切换桌面

【win】 + 【Ctrl】 + 左右键

## 创建新桌面

【win】 + 【Ctrl】 + D

## 锁定桌面

【win】 + L

## 打开或关闭“开始”菜单

【win】

## 打开“操作中心”

【win】 + A

## 将焦点设置到通知区域

【win】 + B

## 在侦听模式下打开 Cortana

【win】 + Shift + C
